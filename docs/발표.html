🔑 왜 공주님이 늘수록 다 부자?

  • 👑 공주님은 운영풀 분배에 안 들어감 — 대화료를 직접 가져감
  • ➕ 공주님 1명 추가 = 풀에 돈만 꽂는 황금알 거위 🪿 (분배자는 그대로)
  • 📈 그래서 공주님 2명→4명에 사장·가드 수익 거의 2배
  • 🔁 추천비(영입 1만·손님 3만)로 모두가 사람을 끌어오는 자가증식

🔄 영업 1회 흐름

  • 1. 오픈 준비 — 위치 확보 · 공지 · 출근체크 · 예약 확인
  • 2. 유치(삐끼) — 홍보 · "OO초대" 손님 기록(→3만)
  • 3. 입장(웨이터) — TC 수납·음료 · 손님 등록 · 공주님 배정
  • 4. 대화 타임 — 20분 타이머 · 팁은 공주님 직접 수령
  • 5. 2차 접수 — 웨이터 단독 · 동의·일정 확인 → 100만
  • 6. 보안(가드) — 진상·강도 대응 · 블랙리스트(밴)
  • 7. 마감 — 정산 · 장부 마감 · 지급
