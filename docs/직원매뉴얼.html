공주님 클럽 직원 매뉴얼

CLIMAX 서버 토킹바 · 우리 모두 다 같이 버는 곳
모든 금액은 게임 내 화폐(RP) 기준. 합법 RP 컨텐츠이며, 현금 거래가 아닙니다.

0. 시작하기

  1. 운영앱 접속: https://gta5-rho.vercel.app
  2. 회원가입 탭 → 사장이 DM으로 준 가입코드 + 이메일/비번 + 이름/전화 입력.
  3. 공주님은 가입 시 추천인(데려온 사람) 전화/닉도 입력.
  4. 로그인되면 사장에게 알리기.
핵심 원칙: 공주님이 늘수록·손님이 늘수록 모두가 번다. 그래서 사람을 데려오는 게 곧 내 수익입니다.

1. 역할별 정산 (얼마 버나)

가격표

항목가격분배
TC(입장료)5만전액 운영풀
대화료25만 / 20분공주 15 / 운영풀 10
2차(외부)100만 / 시간공주 70 / 운영풀 30
팁자유받은 사람 100% 개인 (+@)
주류·담배메뉴별마진(판매−도매) = 출근 전원 N빵

공주님이렇게 법니다

  • 대화료 15만 / 20분 타임 (4시간 풀 = 약 180만)
  • 팁 전액 본인 (+@) — 인기 많으면 기본급보다 더 벌 수도. 상한 없음!
  • 2차 70만 / 건 (본인 선택, 거부권 보장)
  • 주류 마진 N빵 (출근한 날) — 술 팔리면 같이 나눠 가짐
  • 운영풀 지분은 없음(대화료를 직접 가져가니까). 인원수와 무관하게 내 몫 100% 보장

운영스탭(가드·웨이터 겸직) 이렇게 법니다

  • 운영풀 지분 1.0 (출근한 날만!) — 그날 매출(TC·대화료풀·2차풀)을 사장1.2 : 스탭1.0으로 나눔
  • 팁 (웨이터 턴에 받은 것 개인)
  • 주류 마진 N빵 (출근한 날)
  • ⚠️ 출근(체크인) 안 하면 그날 지분 0원. 꼭 출근 버튼 누르기!

삐끼이렇게 법니다 (성과제·한도 없음)

  • 손님 추천 3만 / 명 — 내가 데려온 손님이 이용하면
  • 공주 영입 1만 / 타임 — 내가 데려온 공주가 일하는 내내 계속!
  • 일당·지분 없음. 데려온 만큼 무한대로 법니다.

2. 공주님 업무 가이드

  1. 내 프로필에서 사진 등록 (찌라시 홍보에 사용).
  2. 공주 출근부에서 내가 일할 가용시간을 미리 등록 (예: 21:00~24:00). 본인 것만 등록·수정 가능.
  3. 실제 영업일에 출근 버튼 클릭 → 그때부터 예약을 받습니다. (출근 안 하면 예약 안 잡힘)
  4. 손님과 1:1 대화·접대 RP (20분 타임). 매너로 단골 만들기 = 팁이 곧 내 수입.
  5. 2차는 본인이 원할 때만, 웨이터를 통해서만 잡습니다.
  6. 일 끝나면 퇴근 버튼.
팁은 직접 받아 본인이 보관(앱 장부 미기록). 그 외 모든 돈은 사장에게 모입니다.

3. 운영스탭 업무 가이드

가드 역할과 웨이터 역할을 교대로 봅니다.

출근

  • 스탭 출근부에서 근무시간 등록 + 출근 버튼 (지분 받으려면 필수).

웨이터 턴

  1. 예약판에서 손님 예약 잡기/수정/미루기 (출근한 공주만 선택 가능).
  2. 손님 입장 → TC 수금·음료 제공, 손님 등록.
  3. 대화 타임 진행 누르면 TC·대화료가 자동 거래로 잡힘 (선불).
  4. 2차 접수(웨이터 단독): 공주 동의·일정 확인 → 등록. 2차 시간만큼 그 공주 자리 비움 → 겹치는 예약은 안내대로 조정.
  5. 주류·메뉴 판매: 수금/정산에서 손님+메뉴 수량 입력 → 판매 등록.
  6. 받은 현금은 사장에게 인계.

가드 턴

  • 출입 통제·분위기 관리.
  • 성희롱·진상·시간초과·무전취식은 무관용 → 밴 등록 + 사장/경찰(F3) 신고.
  • ※ 손님을 죽이거나 폭행하면 우리가 범죄자. RP 경고 → 출입거부 → 증거 → 신고 순서로.

4. 삐끼 업무 가이드

  1. 핫스팟에서 홍보, 손님 유치, 새 공주 섭외.
  2. 손님 데려오면 예약 시 추천인(나)으로 등록되게 → 손님추천 3만.
  3. 공주 영입하면 그 공주 가입 시 추천인(나) 등록 → 그 공주 일하는 내내 타임당 1만.
  4. 내 실적은 대시보드 → 추천/영입 실적에서 확인.

6. 필수 규칙

  • 20세 이상만. 성희롱·욕설·패드립·혐오 무관용 → 즉시 출입거부·신고.
  • 모든 활동은 게임 내 RP·게임 내 화폐. 현금 거래 아님.
  • 출근(체크인) 안 하면 정산 못 받음. 허위 출근은 기록으로 추적됨.
  • 팁만 개인, 나머지 모든 현금은 사장에게 집결 → 장부 → 지분 분배(당일).
  • 서버 갈등은 RP 안에서, 마무리 후 F3. 외부(디코 등)로 분쟁 끌고 가지 않기.
자세한 합법성·보안 규정은 통합보고서 참고.
