# Black Out 갱활동 운영앱 — 브레인스토밍 (2026-07-13)

CLIMAX 서버 갱단 **Black Out** 정보부 운영 웹앱 구상. 토킹바(공주님클럽)와는 완전 별개 프로젝트.
현재 상태: **대부님께 약식 제안 전 단계** (`제안/대부님_디스코드_제안.txt`).

## 배경

- 대부(대표)에게 비공식 정보부장 업무 제의받음
- 조직 체계는 잘 잡혀 있음 → 토킹바처럼 웹 시스템으로 지원하면 좋겠다는 구상
- 조직 규칙 중 시스템화 포인트: 상향 보고 체계(6장), 징계 기록(7장), 교전 사후보고 + 증거 영상 확보 의무(9장), 외교 창구 단일화(9장)

## 조직도 (2026-06 기준)

- 대표: 대부
- 부대표: 이흐긴
  - 북부지부장 항문차 → 부장 → 팀장 → 조직원(시화, 데이비드, 짜오미, 임한림, 이명수, 김소신)
  - 남부지부장 이거니 → 부장 → 팀장 → 조직원(송실수, 허춘식, 쥐석영, 박두팔, 김줏대, 예수)
- 정보부: 정보부장(??? ← 내 자리), 정보부원 리아시
- 재정: 차미슬

## 모듈 구상 (4개)

### 1. 정보부 시스템 (핵심 MVP)
- **일반 조직원**: [정보 제공] 탭 — 사진 업로드 + 글 작성으로 찌라시 제보
- **정보부원**: 제보 전체 열람, 카테고라이징, 진위 파악(사실여부), 보안등급 제안
- **정보부장**: 최종 보안등급 확정
  - 보안등급 3단계: 전체공개 / 간부공개 / 정보부장&보스 전용
- **보고서**: 추려진 정보 기반 보고서 작성
- **사회관계도**: 인물 간 관계 시각화
- **정보 거래**: 보스가 판매 가능/불가 결정 → 정보비 구매 장부 기록

### 2. 수금·재정 자동화
- 재정 담당 차미슬 인터뷰(15~30분) 후 스펙 확정
- 수금대장: 자동 취합·관리, 푸시 가능하게
- **의존성: 차미슬 인터뷰 전에는 스펙 못 박음 → 2단계**

### 3. 자산 대시보드
- 조직 자산 현황 시각화
- 향후 돈벌이 사업수단 정리

### 4. 정산 계산기 (편의점털이 등)
- 라운드별 참가자 엔빵 (정산 시점에 라운드별 참가자/횟수 기준 분배)
- 몰아주기 / 사다리타기 모드

## 권한 설계 방향

- 조직원 전체가 쓰되 **권한관리 확실하게** (RBAC)
- 역할 축: 보스 / 간부(부대표·지부장·부장) / 정보부장 / 정보부원 / 일반 조직원 (+재정?)
- 토킹바 앱 RBAC 경험 재활용 가능 (Supabase RLS)

## 결정된 것

- 위치: `gta5/갱활동/` 서브폴더, 토킹바 앱과 완전 분리
- 첫 결과물: 거창한 문서 X → **디스코드 약식 제안 멘트** ("형님~ 이런 거 해드릴 수 있는데 어떠세요")
- 순서: 대부님 반응 보고 → 필요하다는 것부터 개발
