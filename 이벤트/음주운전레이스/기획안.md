# 음주운전 레이스 — 기획안 (v0.1 초안)

> CLIMAX 서버(GTA5 RP) 이벤트. 모든 금액은 게임 내 화폐이며 RP 컨텐츠입니다.
> 주최: 공주님 클럽 (총괄 시진핑)

---

## 1. 한 줄 요약

양주 한 병을 원샷하고, **취한 채로** 맵 최남단에서 최북단까지 완주한다.
죽거나, EMS에 실려가거나, 도착 전에 술이 깨면 탈락. 수단과 방법은 가리지 않는다.

## 2. 왜 재밌나 (기획 의도)

- **취기 = 강제 핸디캡.** 조작 난이도가 급상승해 실력·차량 격차가 희석되고, 사고와 역전이 계속 터진다.
- **관전 재미.** 비틀대는 차, 어이없는 자폭, 마지막 언덕에서 굴러떨어지는 선두. 짧고 강렬하다.
- **우리 사업과 자연스럽게 연결.** 양주 소비 + 브랜드 노출 + 애프터파티(현장 술 판매)로 이어진다.

## 3. 참가와 비용

| 항목 | 내용 |
|---|---|
| 참가비 | 50만 |
| 제공 | 양주 1병 (우리 원가 20만) |
| 실질 참가비 | 30만 (= 50 − 20) |
| 상금 풀 | (참가비 − 양주 원가) 합계에서 상금·운영으로 배분 |

**예시: 10명 참가 시**

| 항목 | 금액 |
|---|---|
| 참가비 수입 | 500만 |
| 양주 원가 | −200만 |
| 순 풀 | 300만 |
| 1등 상금 | 200만 |
| 2등 상금 | 60만 |
| 운영·진행비 | 40만 |

배분 비율은 확정 필요(1등 몰빵 vs 1~3등 분배).

## 4. 규칙 (초안)

1. **출발 전 양주 1병 원샷.** 스탭 입회 하에 참가자 전원 동시 원샷, 육안 확인. 미이행 시 실격.
2. **코스**: 맵 최남단 지정 지점 출발 → 맵 최북단 지정 지점 도착.
3. **수단·방법 무관.** 차량 교체, 지름길, 오프로드, 추격 회피 모두 자유.
4. **즉시 탈락 조건**
   - 사망 후 부활
   - EMS에 발견되거나 이송됨
   - **도착 시점에 취기가 풀려 있음** (도착까지 취한 상태를 유지해야 함)
   - 경찰에 체포됨 (정책 확정 필요, 6번 참고)
5. **먼저 도착한 사람이 우승.**

## 5. 코스

- **출발**: 맵 최남단 (후보: Los Santos 남단 부두 / Elysian Island 남쪽 끝)
- **도착**: 맵 최북단 (후보: Paleto Bay 북단 해안 / Cape Catfish)
- 직선거리 약 8~10km. 도로 주행 기준 대략 8~15분 예상(차량·경로에 따라 편차).

## 6. 리스크와 보완 (반드시 검토)

### 6-1. 치명적: 취기 지속시간 < 완주 시간이면 전원 탈락

"도착까지 취해 있어야 한다"는 규칙은 **취기 지속시간이 완주 시간보다 길어야만** 성립한다.
양주 1병의 취기가 5분인데 완주에 12분 걸리면 **아무도 못 이긴다.**

- **필수 사전 테스트**: 양주 1병 원샷 후 취기가 몇 분 유지되는지 측정.
- **권장 보완책**: 코스 중간에 **체크포인트 1~2곳**을 두고, 그 자리에서 **추가 1샷 의무**.
  - 취기가 갱신되어 규칙이 성립하고,
  - 스탭이 통과를 확인해 부정행위도 막고,
  - 관전 포인트(중간 순위 역전)도 생긴다. **일석삼조라 강력 추천.**

### 6-2. "차량 성능 무관"은 과장

취해도 슈퍼카는 빠르다. 평준화 효과는 있지만 완전하지 않다.

- 대안 A: **차량 등급 제한** (일반 세단·해치백 등급만 허용)
- 대안 B: **동일 차량 지급** (가장 공평, 준비 부담 있음)
- 대안 C: **클래스 분리** (일반부 / 무제한부)

### 6-3. 인증과 부정 방지

- 원샷: 출발선에서 **동시에**, 스탭 육안 확인.
- 사망/EMS: 자진 신고에만 의존하면 구멍. 참가자 **화면 녹화 권장** + 도착선 스탭이 상태 확인.
- 텔레포트·버그·대리 주행: 출발선·중간 체크포인트·도착선에 스탭 배치.

### 6-4. 경찰 개입 정책

"수단·방법 무관"이라면 **경찰 단속을 피하는 것도 게임의 일부**가 되어 재밌다.
다만 **체포 시 탈락**을 명시해야 분쟁이 없다. (사전에 경찰 측에 이벤트 고지 여부도 결정)

### 6-5. 서버 규칙 준수

음주운전은 게임 내 RP 설정이며 서버 규칙 안에서 진행한다. 실제 위법 행위와 무관하다.

## 7. 운영 절차

| 시점 | 할 일 |
|---|---|
| D-3 | 공지·모집 시작, 참가비 접수, 포스터 배포 |
| D-1 | **코스 사전 주행 테스트** (취기 지속시간 측정, 체크포인트 위치 확정) |
| 당일 T-30분 | 출발선 집합, 참가 확인, 양주 지급 |
| T-0 | 전원 동시 원샷 → 스탭 확인 → 스타트 |
| 진행 중 | 중간 체크포인트 스탭 확인(추가 1샷), 도착선 스탭 대기 |
| 종료 | 순위 확정, 시상, 우승자 인터뷰(홍보) |
| 애프터 | 공주님 클럽 애프터파티 (현장 술 판매) |

## 8. 홍보 연계

- 이벤트 포스터(가로·세로) 제작 → 트위터·서버 채널 배포
- 우승자 보상: 상금 + **공주님 클럽 무료 이용권**(TC + 대화료 1타임 등)
- 관전자 대상 현장 주류 판매

## 9. 확정이 필요한 항목

1. 양주 원가 20만이 맞는가? (참가비 50만 대비 실질 30만)
2. **양주 1병 취기 지속시간은 몇 분인가?** (가장 중요)
3. 중간 체크포인트 + 추가 1샷 도입 여부
4. 차량 제한 정책 (무제한 / 등급 제한 / 동일 차량)
5. 경찰 체포 = 탈락으로 확정할 것인가
6. 상금 배분 (1등 몰빵 vs 1~3등)
7. 정원과 조 편성 (권장 8~12명 동시 출발)
8. 출발·도착 지점 정확한 좌표/랜드마크
