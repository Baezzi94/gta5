# 공주님 클럽 (CLIMAX 서버 토킹바 프로젝트)

CLIMAX 서버(GTA5 FiveM 인생서버) 토킹바 RP 사업 + 운영 웹앱 저장소.

## 구성

```
gta5/
├─ docs/                         # 기획·발표·규정 문서
│  ├─ 사업계획서.md               # 사업계획서 원본(MD)
│  ├─ 계획서_위키.html            # 사업계획서 위키형(팀 공유용)
│  ├─ 합법성_및_보호규정.md        # 합법성·구성원 보호 규정 원본(MD)
│  ├─ 합법성_및_보호규정_위키.html # 위 문서 위키형
│  ├─ 공주님클럽_통합보고서.html   # 사업계획 + 합법성 탭 통합본
│  └─ 발표.html                   # 발표용 슬라이드
└─ (app/)                        # 운영 웹앱 (예정: 예약/밴/손님/출근부/정산)
```

## 운영 웹앱 (개발 예정)

- 스택: Supabase(인증·DB·권한) + 웹 프론트엔드
- 핵심 기능: 손님 관리 / 밴 관리 / 예약 관리 / 공주님 출근부 / 거래 입력 / 정산 자동배분 / 대시보드
- 권한(RBAC): 사장(전체) · 운영스탭(예약·밴·손님·거래) · 삐끼(추천) · 공주님(본인 조회)

## 버저닝

- `main` 브랜치 = 안정 버전
- 기능 단위로 커밋, 의미 있는 단계마다 태그(`v0.1.0` 등)
- 롤백: `git checkout <태그/커밋>` 또는 `git revert <커밋>`

## 규칙

본 프로젝트의 모든 활동·거래는 게임 내 RP와 게임 내 화폐로만 이루어지며, CLIMAX 서버 법률·운영정책을 준수한다. (자세한 내용: `docs/합법성_및_보호규정.md`)
